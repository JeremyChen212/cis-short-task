Hybrid Applications

Introducing Hybrid Applications! Hybrid applications are a convenient solution for developing cross platform applications that can be used in the browser, as well as natively as an IOS or Android mobile app.

The 3

The 3 frameworks used to make hybrid applications that will be discussed is...
React Native, Flutter , & PhoneGap

Similarities

Each of these frameworks support cross-platform development, meaning a single codebase can be used to build apps for both IOS and Android platforms.

Framework Overview

PhoneGap is a framework utilizes web technologies such as HTML, CSS, and JavaScript. It provides cross-platform development by wrapping the app's web code into a native WebView container, allowing access to device features. Because it relies on all these web technologies, it does not provide the same level of performance as fully native apps.

ReactNative, developed and used by Facebook, allows developers to build mobile applications using the language Javascript, one of the most popularly used languages in the world. This framework offers a range of features from code reusability (components), fast performance, and live reloading.

Flutter, developed by Google, uses the language Dart, an object-oriented programming language built for the purpose of creating applications. Some features this framework offers are fast and customizable UI rendering, pre-built UI components, hot rload for quick development changes, and a widget based UI architecture.
